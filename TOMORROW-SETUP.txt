## 1. Google Calendar MCP (15-20 mins)

**What it enables**: Calendar integration, therapy session reminders, automatic todo extraction from calendar

### Steps:
1. Go to https://console.cloud.google.com/
2. Create/select a project
3. Enable **Google Calendar API**
4. Create **OAuth 2.0 Client ID** → Desktop application
5. Download JSON credentials
6. Move to: `/Users/jamescherry/.config/google-mcp/credentials.json`
   ```bash
   mv ~/Downloads/client_secret_*.json /Users/jamescherry/.config/google-mcp/credentials.json
   ```
7. In Claude Code, run: `/mcp`
8. Authenticate with Google Calendar

**Test**: Run `/mcp` - should show google-calendar as "connected"

## 2. Apple Health Integration (Research needed)

**What it enables**: Track mental/physical health metrics (sleep, activity, heart rate variability)

### Options to explore:

**Option A: Apple Shortcuts**
- Create shortcuts that export Health data to text files
- Point coaching system to read those files
- Similar to how Diarium exports work

**Option B: MCP Server**
- Check if Apple Health MCP server exists
- Install if available
- https://github.com/modelcontextprotocol/servers (check registry)

**Option C: Manual export**
- Export Apple Health data regularly
- Save to `~/Documents/Claude Projects/claude-shared/health-data/`
- Parse manually

**Recommended**: Start with Option A (Shortcuts) - quickest to set up

### What to track (examples):
- Sleep hours
- Sleep quality
- Steps
- Heart rate variability (HRV) - good anxiety indicator
- Mindful minutes
- Active energy
